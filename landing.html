Summary: Latitude vs. X

The purpose of this project was to ultize weather data and search for correlations with latitude. We pulled this data from the OpenWeatherMap API and created a dataset of over 500 cities. Using Matplotlib, we created scatter plots that compared latitudes of the cities with four different weather conditions: temperature, humidity, cloudiness, and wind speed. This site was created to display the visualizations and analysis from this project. Some visualization plots indicated a positive or negative correlations; however, some indicated more complex causes for weather conditions than latitude.
